什么是 HTTPS ?一句话，HTTPS = HTTP + SSL。HTTPS 并不是一个全新的协议，而是在 HTTP 的基础上，通过 SSL 增加了一层加密协议，从而大大增加了 HTTP 协议的安全性。

所以在正式了解 HTTPS 之前，我们需要先了解 HTTP。
